# package.json

Node.js package manifest. Lists dependencies, scripts, and project metadata.

## Example

```json
{
  "name": "structures",
  "version": "0.0.0",
  "private": true,
  "license": "MIT",
  "scripts": {
    "ng": "ng",
    "start": "ng serve",
    "build": "ng build",
    "watch": "ng build --watch --configuration development",
    "test": "ng test"
  },
  "dependencies": {},
  "devDependencies": {}
}
```
